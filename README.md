# Автоматизированное тестирование API Swagger PetStore

Проект демонстрирует работу по автоматизации тестирования API [Swagger PetStore](https://petstore.swagger.io/)

## Обзор проекта

Проект находится в процессе развития, постепенно дополняется и оптимизируется

### Уже реализовано

- Создана основная структура проекта, настроены зависимости (pom.xml), базовая конфигурация (RestConfig)
- Классы модели для /pet (Pet, PetCategory, PetStatus, PetTag)
- Класс клиента для /pet (PetClient)
- Настроена инфраструктура для тестов
- Настроена генерация тестовых данных (PetFactory)
- Основные позитивные CRUD тесты для /pet (PetTests)
- Позитивный параметризованный тест для поиска питомцев по статусу
- В тестовом классе добавлены вспомогательные методы для уменьшения дублирования кода 
- Добавлено полное логирование запросов и ответов

### Структура проекта

```text
src
├── main
│   └── java
│       └── api
│           ├── clients       # Классы клиентов для работы с API 
│           ├── models        # Модели данных 
│           ├── spec          # Спецификации запросов и ответов 
│           └── config        # Конфигурация REST (RestConfig)
└── test
    └── java
        └── tests
            ├── BaseTest      # Базовый тестовый класс с общими настройками
            └── ...           # Тесты 
```
